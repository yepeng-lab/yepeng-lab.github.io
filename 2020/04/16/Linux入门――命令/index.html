Linux入门――文件处理命令

发表于 2020-04-16 | 分类于 Linux |

文件处理命令

ls(list)

执行权限:所有用户

ls 显示当前目录文件

  • -a(–all)：显示所有文件

  • -l(long)(显示当前文件下文件的属性)

  • -h(humanize):将文件大小以k、mb这样的形式表现出来

  • -i：显示文件的i结点

mkdir(make directories)：创建目录

权限：所有用户

  • -p：递归创建

cd(change directory)

权限：所有用户

pwd(print working directory)：显示当前目录路径

权限：所有用户

rmdir(remove empty direcories)：删除空白目录

权限: 所有用户

cp(copy)：复制

cp [源文件目录] [目标文件目录]

  • rp

  • r：拷贝目录

  • p：保留文件属性

在复制的同时也可以改名

mv(move)：剪切、移动,也可以改名

与cp 大同小异，不需要-r

rm(remove)：删除

  • -rf

  • -r：删除目录

  • -f：强行执行

touch：创建空文件

cat: 查看文件内容

  • -n：显示行号

tac：反向显示

more：分页显示文件内容

  • 冒号：浏览

  • f、空格：翻页浏览

  • 回车：一行一行浏览

  • q：退出

less：可以向上翻页

  • 和more一样

  • pageup：向上翻页

  • 上箭头：向上换行

  • /：搜索:按n往后找

head：查看文件前几行

  • -n：显示前n行，默认前几行

tail：后几行

  • 与head一样

  • -f：动态显示，如果有变动就会显示

ln：链接

  • -s创建软链接

ln -s file1 file2：生成file1的软链接file2

软链接类似于Windows的快捷方式

ln file1 file2：生成file1的硬链接file2

硬链接可以同步更新，源文件即使丢失，硬链接也能生效

如何区别硬链接和源文件

注意：

  1. 硬链接不能跨分区

  2. 硬链接不能针对目录使用
