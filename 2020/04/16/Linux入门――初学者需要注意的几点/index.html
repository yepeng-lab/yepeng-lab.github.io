Linux入门――初学者需要注意的几点

发表于 2020-04-16 | 分类于 Linux |

Linux严格区分大小写

不像Windows那样，Linux系统严格区分大小写

Linux中所有信息靠文件保存

Windows中，有些信息是储存在文件的，包括硬件，如果想要永久生效，就必须以文件的形式保存

Linux不考扩展名区分文件类型

虽说如此，但是Linux并非没有文件扩展名，一般为了区分，使用方便，写清楚压缩包名：

  • 压缩包： .gz、.bz2、.tar.bz2、.tgz

  • 二进制软件包：*.rpm

  • 网页文件：.html、.php

  • 脚本文件：*.sh

  • 配置文件：*.conf
    但是这些不是必需的

Linux所有的存储设备都必须挂载之后才能使用，包括硬盘、U盘、光盘

挂载：相当于Windows的分配盘符（分区）

Windows所有的程序不能直接在Linux安装使用

Linux各目录信息

目录名称 目录作用
/bin/(binary,二进制的) 存放系统命令，所有用户都可以使用
/sbin/ 只有root用户才能使用的和环境设置相关命令
/usr(Unix System Resource)/bin/ 在单用户模式下不能执行
/usr/sbin 存放根文件系统不必要的系统管理命令
/boot/ 系统启动目录，保存系统启动相关的文件
/dev/ 设备文件保存位置。
/ect/ 配置文件保存位置
/proc/ 虚拟文件系统，数据保存到内存当中
/sys/ 虚拟文件系统，主要保存于内核相关的信息
/root/ 超级用户的家目录
/srv/ 服务数据目录，一些系统服务启动以后，可以保存所需要的数据
/tmp/ 临时目录，系统存放临时文件的目录
/usr/ 系统软件资源目录
var 动态数据保存位置

##　服务器注意事项

  • 远程服务器不允许关机，只能重启
  • 重启时应该关闭服务
  • 不要在访问高峰运行高负载命令
  • 远程配置防火墙时不要把自己踢出服务器
  • 指定合理密码规范并定期更新
  • 合理分配权限
  • 定期备份
