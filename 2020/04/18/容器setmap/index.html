容器set

简介

set/multiset是以rb_tree为底层结构，因此以key(value)为依据自动排序。

set/multiset提供遍历操作以及iterator，按正常规则(++ite)遍历，以便能获得排序状态

无法使用iterator改变元素值

set和multiset调用的函数不同

set in GNC

set所有的操作，都是调用红黑树(t)的操作，从这个意义上来看，set未尝不是container adapter。

set in VC6

VC6不提供identity，那么set和map如何使用RB-tree
