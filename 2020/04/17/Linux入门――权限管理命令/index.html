Linux入门――权限管理命令

发表于 2020-04-17 | 分类于 Linux |

chmod(Change the permissions mode of a file)：更改文件权限

只有所有者和管理员才能更改

  • [{ugoa}{+-=}{rwx}][文件和目录]

如：chmod u+x Japanvovestory.list

chmod g+w,o-r,u=rwx filename

但是在Linux中使用最多的是数字：

r – 4

w – 2

x – 1

  • chmod n filename

  • R：递归修改

rwx权限

代表字符 权限 对文件的含义 对目录的含义
r 读 查看文件内容 列出目录中的内容(ls)
w 写 修改文件内容 在目录中创建、删除文件(touch/mkdir/rmdir/rm)
x 执行 可以执行文件 可以进入目录(cd)

chown(change file ownership)：改变所有者

  • 所有者、管理员

  • chown [用户][文件或目录]

chgrp(change file group ownership)：改变所属组

  • chgrp [用户组][文件或目录]

umask(the user file-creation mask)：显示默认权限

  • -S：以rwx形式显示新建文件默认权限

注意：为了一定的安全性，创建的文件默认没有执行权限

注意：直接输入umask，得到的值，与777进行异或运算(相同为零)

注意：文件默认没有执行权限

  • umask xyz 改变默认权限
