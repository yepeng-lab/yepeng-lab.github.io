Linux入门――文件搜索命令

发表于 2020-04-17 | 分类于 Linux |

find：文件搜索

  • find [搜索范围][匹配条件]

  • -name：根据文件名搜索

    • find / -name filename ：模糊搜索

    • find / -name filename* :以filename开头的文件

    • find / -name file???? :　以file开头，文件名一共8个字母

    • -iname：不区分大小写

  • -size: 指定文件大小:以数据块（1数据块=512字节=0.5k）

    • +:大于

  • -user：根据所有者查找

  • -ctime：按照时间

    • amin：访问时间access

      • cmin：文件属性change

      • mmin：文件内容modify

    • find / -ctime (+-=)number

  • -a:表示and，同时满足：find / -size +46432 -a -size -2012313

  • -o:表示or

  • -type 搜索文件类型

    • find / -name file* -a -type f/d/l

  • inum 软链接

  • -exec/-ok 命令 {} ;对搜索结果执行

技巧：i节点有大用处
