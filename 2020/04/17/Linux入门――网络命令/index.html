Linux入门――网络命令

发表于 2020-04-17 | 分类于 Linux |

write

  • 给用户发消息，以Ctrl+D保存结束

  • write<用户名>

wall

  • write all

ping

  • ping ip

  • -c+n ，pingn次

ifconfig/ip addr

mail

  • mail [发信用户名]

last

  • 查看所有用户登录信息

lastlog

  • lastlog –u uid 显示指定用户最后登录时间

traceroute 探测到指定网站的路径

我在这步的时候显示未知的命令，只要执行以下命令就行：

netstat

  • 显示网络相关信息

  • -t 查询TCP协议

  • -u UDP协议

  • -l 监听

  • -r 路由

  • -n 显示ip和端口号

例子：
netstat -tlun 查看本机监听的端口

netstat -an 查看本机所有网络连接

netstat -rn 查看本机路由表

setup：在其他版本的Linux不存在

自动配置网络

mount 挂载

  • mount[-t 文件系统] 设备文件名 挂载点

umount 卸载
